# BUILD Phase

**Task:** {user_query}

---

## Purpose

Orchestrate agents to create artifacts.

## Actions

1. **Select Agents:** Which agents are needed for this task?
2. **Prepare Context:** What information do agents need?
3. **Invoke Agents:** Execute agent workflows
4. **Collect Artifacts:** Gather outputs from agents

## Available Agents

- **analysis:** Decompose problems, identify patterns
- **clarification:** Resolve ambiguities with user
- **generation:** Create code and artifacts
- **research:** Investigate options and gather knowledge
- **synthesis:** Integrate findings into recommendations
- **validation:** Verify artifacts against criteria

## Output

Artifacts ready for EXECUTE phase.

**Then:** Proceed to EXECUTE phase
