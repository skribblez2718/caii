# THINK Phase

**Task:** {user_query}

---

## Purpose

Apply chain of thought reasoning to break down the problem.

## Actions

1. **Decompose Problem:** Break into smaller, manageable sub-problems
2. **Identify Dependencies:** What must happen before what?
3. **Consider Approaches:** What strategies could solve each sub-problem?
4. **Evaluate Trade-offs:** What are the pros/cons of each approach?

## Output

A reasoned analysis that feeds into PLAN phase.

**Then:** Proceed to PLAN phase
